## DMIT 是什么，为什么值得考虑

DMIT（dmit.io）是一家从 2018 年跑起来的 VPS 服务商，美国纽约注册，背后是华人团队，有中文客服，支持支付宝付款。

**DMIT 最核心的卖点就一个：线路质量。** 全系标配 AMD EPYC 处理器 + NVMe SSD，本身硬件就不算差。但让它在圈子里出名的，是它对中国方向的路由优化——直接跟三大运营商对接 CN2 GIA、CMIN2、CMI 等精品线路，晚高峰也基本扛得住。

日常价格比便宜货贵一截，但促销期间性价比会明显拉升。而且 DMIT 有个不常见的良心设计：流量超限之后不关机，直接切到限速模式继续跑，对大流量场景很省心。

IP 被墙之后可以每 15 天免费换一次，这点对跑代理的人很实用。退款政策也算清晰：3 天内流量不超 30GB 可全额退，30 天内按剩余价值退。

## DMIT 产品线是什么结构

看 DMIT 官网第一眼容易懵，产品命名比较绕，先把三条线路的区别说清楚：

**Premium（Pro 系列）**：旗舰线路。电信联通去程 CN2 GIA，移动 CMI 去程，三网 CN2 GIA 回程。晚高峰最稳，但价格也最高。适合对大陆访问速度要求高的场景——建站、API 服务、代理节点，电信用户尤其合适。

**Eyeball（EB 系列）**：性价比路线。洛杉矶 EB 走 CMIN2 三网回程，香港和东京 EB 走 CMI 优化回程。比 Pro 便宜一档，对联通移动用户更友好，适合预算有限但不想将就的人。

**Tier 1（T1 系列）**：国际优化线路，不做中国大陆专项优化，但流量最大，AMD EPYC 9005 平台硬件最新，超量不关机直接限速，适合国际业务落地或有自建中转方案的用户。

三个机房也各有侧重：
- **洛杉矶 LAX**：产品线最全，Premium / Eyeball / Tier 1 / Premium Secure 四条线路都有
- **香港 HKG**：距离大陆近，延迟通常在 20-50ms，是国内用户建站的首选
- **日本东京 TYO**：亚太布局，CN2 GIA + CMI 优化，适合日本 IP 或东南亚业务

## 常见问题

**Q：DMIT 的优惠码会过期吗？**

会。部分是有明确截止日期的活动码，部分是长期有效的循环码。上面表里的几个当前都在使用周期内，但具体状态以购物车验证结果为准。过了有效期或不适用当前套餐，页面会有提示。

**Q：年付和月付差距有多大？**

DMIT 年付本身比月付便宜不少，再叠加年付专属优惠码，差价非常可观。比如 LAX T1 TINY 月付大约 $6.9，年付基础价已经低于 12 个月月付总价，再用 7 折码，长期持有成本直接低到月均两三刀。

**Q：库存不够怎么办？热门套餐经常缺货？**

对，这个是 DMIT 的客观问题。热门套餐（尤其是特价机型）补货后可能几小时就售罄。进官网能看到套餐是否 In Stock，看到合适的不要等。

**Q：流量超了会停机吗？**

不会直接停机。T1 系列超量后会自动切到限速模式继续跑，香港和东京 WEE 套餐超量限速 50Mbps，洛杉矶 WEE 超量限速 100Mbps，下个月流量重置后自动恢复正常速度。Pro 和 EB 系列超量后会暂停，要续费才能恢复。

**Q：退款政策怎么说？**

购买 3 天内、流量使用不超 30GB 可以全额退款，退到账户余额免手续费，退回原支付方式可能有手续费。30 天内按剩余价值可部分退款。不确定适不适合，可以先买最小套餐跑一个月测测线路再决定。

**Q：支付方式支持哪些？**

支持支付宝、PayPal、信用卡，对国内用户来说付款没有门槛。
